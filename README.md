# 📊 Sales Prediction using Machine Learning

An advanced Machine Learning project built using **Python, Scikit-learn, Streamlit, Pandas, NumPy, and Plotly** to predict product sales based on advertising investments across multiple marketing platforms such as **TV, Radio, and Newspaper**.

This project combines **Data Analysis, Data Visualization, Machine Learning, and Interactive Dashboard Development** to help businesses understand advertising performance and make data-driven marketing decisions.

The system analyzes advertising budgets, identifies patterns affecting sales, trains a predictive model, and provides real-time sales predictions through a modern interactive dashboard.

# 📈 Key Business Insights

- TV advertising has the strongest impact on product sales.
- Higher advertising investments generally increase sales performance.
- Radio advertising significantly improves sales predictions.
- Newspaper advertising contributes the least compared to other platforms.
- Sales show a strong positive relationship with TV and Radio marketing.
- Businesses can optimize marketing budgets using predictive analytics.
